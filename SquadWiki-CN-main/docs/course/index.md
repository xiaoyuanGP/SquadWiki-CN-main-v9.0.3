# 战术小队游戏指南

《战术小队》(Squad)是一款高度拟真的大型多人在线战术射击游戏，具有复杂的指挥系统、物资补给机制和团队协作要求。本指南将帮助您系统掌握游戏核心玩法。

## 新手入门路径

1. **基础设置** - 优化游戏画面和操作设置
2. **基础操作** - 掌握移动、射击和通讯
3. **团队协作** - 了解小队配合基本规则
4. **战斗技巧** - 学习基本战术和生存技巧

> 重要提示：游戏有严格的友军伤害惩罚机制，请务必注意敌我识别！
